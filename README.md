## Install

```zsh
git clone https://github.com/johnhodge/.dotfiles_mbp.git && cd .dotfiles_mbp && ./install
```

## Manual steps

- Set macOS color mode
- Turn off Spotlight shortcut
- Install synology drive client
- Configure Alfred
- Install fonts
